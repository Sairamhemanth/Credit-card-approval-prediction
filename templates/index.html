About This Project

This Credit Card Approval Prediction System is an AI-powered web application developed to assist financial institutions in evaluating credit card applications efficiently. The system combines Banking Eligibility Rules with a Decision Tree Machine Learning model to analyze demographic, employment, financial, family, and housing information before generating an approval decision.

Along with the prediction result, the application also provides meaningful reasons and recommendations, making the decision transparent and easier to understand.
